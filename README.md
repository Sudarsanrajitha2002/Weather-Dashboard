## Overview 
Real-time weather application with location-based forecasts and interactive visualizations. 
 
## Features 
- Current weather conditions 
- 7-day forecast 
- Location-based weather 
- Weather alerts 
- Interactive charts 
 
## Tech Stack 
- **Frontend:** React.js 
- **API:** OpenWeatherMap API
